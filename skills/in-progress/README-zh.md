# In Progress

仍在开发中的 skills。它们尚未准备好发布——预计会有粗糙的边缘、 breaking changes 和被放弃的实验。在它们毕业到稳定的 bucket 之前，它们被排除在 plugin 和顶层 README 之外。

- **[decision-mapping](./decision-mapping/SKILL.md)** — 将松散的想法转化为调查 ticket 的有序图，然后逐个驱动它们到解决。 User-invoked。
- **[loop-me](./loop-me/SKILL.md)** — 在多个 sessions 中将自己 grilling 为可实现的 workflow specs，使用当前目录作为有状态的 workspace。 User-invoked。
- **[review](./review/SKILL.md)** — 沿两个平行轴审查自固定点以来的变更：**Standards**（diff 是否遵循 repo 的编码标准？）和 **Spec**（diff 是否忠实地实现了源 issuse/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 将文章塑造成 beat 的旅程，选择你自己的冒险风格。选择一个起始 beat，只写那个 beat，然后转向下一个，直到文章达到自然结尾。
- **[writing-fragments](./writing-fragments/SKILL.md)** — Grilling session，挖掘你的 fragments ——异质的写作碎片——并将它们追加到一个文档中，作为未来文章的原始材料。
- **[writing-shape](./writing-shape/SKILL.md)** — 获取原始材料的 markdown 文件，逐段将其塑造成文章，每步讨论格式选择。
