# Triage Labels

Skills 以五个 canonical triage roles 来表述。此文件将这些 roles 映射到此 repo issue tracker 中使用的实际 label 字符串。

| mattpocock/skills 中的 Label | 我们 tracker 中的 Label | 含义 |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待报告者提供更多信息 |
| `ready-for-agent` | `ready-for-agent` | 完全指定， AFK agent 可拾取 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会被执行 |

当一个 skill 提及一个 role 时（例如"apply the AFK-ready triage label"），使用此表中的相应 label 字符串。

编辑右列以匹配你实际使用的任何词汇。
